"Youme" Demos

The pages below demonstrate various components provided by this package.

  • "Passthrough": Relay messages from a single input to a single output.
  • "Multi-Passthrough": Relay messages from one or more inputs to one or more outputs.
  • "Message Monitor": Display MIDI messages received from one or more inputs.
  • "MIDI Console": Convert MIDI messages between hex and JSON, send them to an output.
  • "Quarter Frame Receiver": Update a local clock based on external quarter frame MTC messages.
  • "Quarter Frame Sender": Send quarter frame MTC messages to an external receiver to control its clock.
